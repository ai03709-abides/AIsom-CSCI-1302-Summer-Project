# AIsom-CSCI-1302-Summer-Project
Summer 2026 Passion Project for the CSCI 1302 course. The entry is a simple global distance calculator using coordinates.

Run the program to start the application.

Type in the coordinates for two points whose distance is inquired. Seperate the latitude and the longitude by a comma and a space.
Only numbers and the middle characters are accepted. Any deviation will result in a visual request to update the entries.

Hit the calculate button to see the result, visible next to the distance label.
